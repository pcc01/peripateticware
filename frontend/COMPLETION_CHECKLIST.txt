🚀 QUICK START
────────────────────────────────────────────────────────────────────────────

1. Install & Setup:
   $ cd /home/claude/peripateticware-frontend
   $ npm install
   $ cp .env.example .env.local

2. Start Development:
   $ npm run dev
   -> Opens http://localhost:5173

3. Run Tests:
   $ npm run test

4. Start Backend:
   $ cd /path/to/backend
   $ python -m uvicorn main:app --reload --port 8010

5. View Components:
   $ npm run storybook
